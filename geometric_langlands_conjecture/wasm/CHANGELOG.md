## [0.1.0-alpha] - 2025-01-19

### Added
- Initial alpha release
- Basic reductive groups: GL(n), SL(n), Sp(2n), SO(n)
- Simple automorphic forms: cuspidal and Eisenstein series
- Elementary Galois representations: Artin and Weil-Deligne
- Basic L-function computations
- WebAssembly bindings for browser use
- Interactive web demo
- Basic examples demonstrating working features

### Known Limitations
- Limited to small rank groups (dimension ≤ 10)
- Simplified correspondence implementation
- No GPU acceleration
- Limited test coverage
- Some theoretical features not yet implemented

### Documentation
- README with alpha warnings
- ROADMAP for future development
- Examples showing current capabilities
- API documentation (basic)
